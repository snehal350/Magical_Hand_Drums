# Magical_Hand_Drums
"Hand-tracking virtual drum kit using Mediapipe, OpenCV and Pygame- play drums with finger counts."
## Demo
![demo animation](assets/demo.gif)

## Features
- Detects left & right hand finger counts.
- Maps finger count to drum pads (Kick, Snare, HiHat, Tom, Crash).
- Procedural drum-like sounds (no external samples).
- Particle visual effects and webcam "magic mirror".
